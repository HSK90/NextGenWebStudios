# NextGenWebStudios

Proyecto: Mission Start: Auditoría de Rendimiento y Estandarización del Workspace en "NextGen Web Solutions"

## Setup inicial

1. Instalar Node.js y npm en el equipo.
2. Ejecutar `npm install` en la carpeta del proyecto.
3. Ejecutar `npm run lint` para validar el código.
4. Ejecutar `npm run format` para formatear todos los archivos.
5. Iniciar el microservicio con `npm start`.

## Estructura creada

- `package.json` con scripts de inicio, lint y formateo.
- `.eslintrc.json` para estandarizar reglas de Node y JavaScript.
- `.prettierrc` para formateo consistente.
- `src/index.js` es el punto de arranque del microservicio.
- `.vscode/extensions.json` y `.vscode/settings.json` para un workspace listo para desarrollo.
- `AUDIT.md` con guía de auditoría DevTools.
